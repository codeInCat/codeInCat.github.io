this的几种用法

this是JavaScript语言的一个关键字。随着函数使用场合的不同，this的值会发生变化。JavaScript中的this指针是一个动态的变量，一个方法内的this指针并不是始终指向定义该方法的对象的。

this关键字虽然会根据环境变化，但是它始终代表的是调用当前函数的那个对象。这就引出了JS中函数调用的问题。在JS中调用函数的模式可以分为4种： 方法调用模式、函数调用模式、构造器调用模式、apply调用模式。

方法调用模式

当函数被当做某个对象的方法调用时,此时函数内部的this指向调用他的对象; 

    function sayName(){
        alert(this.name);
    }
    var name = 'window';
    var obj = {
        name: 'obj',
        sayName: sayName
    }
    obj.sayName();//obj

函数调用模式

当函数以该模式调用时,this指向的是window对象;

    function sayName(){
        alert(this.name);
    }
    var name = 'window';
    var obj = {
        name: 'obj',
        sayName: sayName
    }
    sayName();//window

构造函数调用模式

如果在一个函数前面加上new关键字来调用，那么就会创建一个连接到该函数的prototype成员的新对象，同时，this会被绑定到这个新对象上。即this指向该构造函数的实例对象;

    function Dog(name,age){
        this.name = name,
        this.age = age, 
        this.sayName = function(){
            alert(this.name)
        }
    }
    var xiaobai = new Dog('xiaobai',2);
    xiaobai.sayName();//xiaobai

apply/call调用模式

所有函数对象都有两个方法：apply和call，这两个方法可以让我们构建一个参数数组传递给调用函数，也允许我们改变this的值。
注意:非严格模式下,apply未传参或第一个参数是null或undefined时,this指向的是window

    var name = "window";
    var person = {
        name: "xiaobai"
    };
    function sayName() {
          console.log(this.name);
    }
    sayName();    //window
    sayName.apply(person);   //xiaobai
    sayName.apply();    //window
