Markdown语法学习笔记

标题

标题是每篇文章都需要也是最常用的格式，在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。

以此类推，总共六级标题，建议在#号后加一个空格，这是最标准的Markdown 语法。

列表

列表的显示只需要在文字前加上 + 或- 或 * 即可变为无序列表

  • item1
  • item2
  • item3

有序列表则直接在文字前加1. 2. 3.

  1. 海上钢琴师
  2. 天堂电影院
  3. 西西里的美丽传说

引用

如果你需要引用一小段别处的句子，那么就要用引用的格式.只需要在文本前加入 > 这种尖括号（大于号）,可以嵌套

最美的不是下雨天,而是在陪你躲过雨的屋檐

不能说的秘密

周杰伦

换行

只需在当前行结尾加两个空格,如果直接换行会显示为一个空格

在当前行的结尾加2个空格  
这行就会新起一行

图片与链接

插入链接与插入图片的语法很像，区别在一个 !号

链接为：[]()

穿越时空的少女

图片为：![]()

粗体与斜体

用两个*包含一段文本就是粗体的语法，用一个*包含一段文本就是斜体的语法。

疏影很邪水清浅 暗香浮动月黄昏
